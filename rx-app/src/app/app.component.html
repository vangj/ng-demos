Purpose

Shows how to use RxJS with some functions.

  • map, tap, flatMap, mapTo, scan, filter, debounceTime
  • catchError
  • mergeMap transformation
  • from: Converts almost anything to an Observable. creation
  • switchMap: Maps each value to an Observable, then flattens all of these inner Observables. transformation

  • interval: Emits ascending numbers, one every second (1000ms) up to the number 3 creation
  • retry
  • share multicast
  • zip
